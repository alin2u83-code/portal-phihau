Migrarea Button — sprint 1

ÎNAINTE (hardcodat)
primary: "bg-indigo-600
  hover:bg-indigo-700
  text-white"

Culoarea nu se schimbă niciodată — indiferent de temă.

DUPĂ (urmează tema)
primary: {
  backgroundColor:
    'var(--t-primary)',
  color:
    'var(--t-primary-fg)'
}

Hover prin CSS: filter: brightness(0.9) pe :hover — fără JS.

⚠ danger/success/info/warning rămân cu Tailwind hardcodat — nu urmează tema (culori semantice fixe).
