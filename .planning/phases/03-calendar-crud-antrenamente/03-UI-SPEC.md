## Accessibility Contract

- All interactive calendar cells: `cursor-pointer` + focusable via keyboard (tabIndex implied by div onClick)
- Buttons in DayPanel: minimum 40px height, `touch-manipulation` class for mobile
- Modal uses `Modal` from `components/ui.tsx` which handles Escape key, focus trap, portal in `document.body`
- Icon-only buttons (ConfirmButton with TrashIcon only): must have accessible label via title or aria-label: `aria-label="»Шterge antrenament"`
- Badge text is visible in all badge variants (color-mix ensures sufficient contrast against dark background)
- `font-mono` for time display ensures consistent character width alignment in rows
