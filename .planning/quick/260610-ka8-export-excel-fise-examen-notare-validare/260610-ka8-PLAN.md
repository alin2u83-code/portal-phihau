# Quick Task 260610-ka8: Export Excel Fișe Examen

## Goal
Add Excel export for 2 exam formats:
1. **Fișă Notare** (federation format) — blank template before exam, grade/notes cols empty
2. **Fișă Validare** (local format) — blank template before exam, Admis/Contrib/Obs cols empty

Both: comisia preloaded from sesiune.comisia[], editable in modal before download.

## Tasks

### T1: Create exportExamenService.ts
**File:** `services/exportExamenService.ts`
**Action:** Create service with two functions using xlsx library:
- `generateNotare(sesiune, inscrieri, grade, comisie)` → triggers .xls download
- `generateValidare(sesiune, inscrieri, grade, comisie)` → triggers .xls download

**Format 1 (Notare):**
- Header rows: federation title, club, date, localitate, comisia (3 rows)
- Column headers rows 14-15: Nr, NUME/PRENUME (split), Club, Grad sustinut, Tehnica, Doc, Song, Thao, Nota generala
- Data rows: nr, sportiv_nume, sportiv_prenume, clubNume(blank), grad_sustinut — all note cols EMPTY
- Sorted by grad ordine descendent

**Format 2 (Validare):**
- Header rows: federation title, club, date, localitate, comisia
- Column headers row 9: Nr, Nume/Prenume Sportiv, Gradul sustinut, Admis/Respins, Contributia, Obs
- Data rows: nr, "Nume Prenume" combined, grad_sustinut — Admis/Contrib/Obs EMPTY

**done:** `services/exportExamenService.ts` created and compiles
